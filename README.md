**Glitch Fall Game – OpenCV**
Glitch Fall is a simple falling block puzzle game (Tetris-style) developed using **Python, OpenCV, and NumPy**.
The game uses a 2D grid system where blocks fall automatically, and the player arranges them to clear rows and score points.
** Technologies Used**
- Python 3
- OpenCV (cv2)
- NumPy
**Features**
- Falling block gameplay
- Row clearing
- Score display
- Level-based speed increase
- Keyboard controls
- Glitch visual effects
- Single Python file implementation
** Controls**
A – Move Left  
D – Move Right  
W – Rotate Block  
ESC – Exit Game
